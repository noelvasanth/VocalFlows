# VocalFlows AI
VocalFlows AI is a co-creative and suggestive AI that suggests vocal melodies or flows given an input lyric or phrase

Github repository is a work in progress
### Dataset

1. The top 500 songs dataset can be downloaded from [here](https://archive.org/details/top-500-songs-dataset).
2. The Phoneme Audio Pairs dataset can be found [here](https://github.com/noelvasanth/VocalFlows/blob/main/Datasets/top4000_database.pickle)
